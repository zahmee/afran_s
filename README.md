# أفران الجنوب - موقع ستاتيك

موقع دعائي وتعريفي لأفران الجنوب، مبني بـ HTML/CSS/JavaScript خالص.

## هيكل المشروع

```
afran_site/
├── index.html          # الصفحة الرئيسية
├── css/
│   └── style.css       # التنسيقات
├── js/
│   └── main.js         # السكريبتات
├── images/             # مجلد الصور
│   └── README.txt      # دليل أسماء الصور المطلوبة
└── README.md
```

## الأقسام

- **الرئيسية (Hero)** - قسم ترحيبي
- **من نحن (About)** - تعريف بالمخبز
- **منتجاتنا (Products)** - شبكة المنتجات الرئيسية
- **معرض الصور (Gallery)** - معرض صور المنتجات
- **اتصل بنا (Contact)** - معلومات التواصل

## التشغيل

افتح ملف `index.html` مباشرةً في المتصفح، أو شغّل سيرفر محلي:

```bash
python -m http.server 8000
```

ثم افتح: http://localhost:8000

## إضافة الصور

ضع الصور في مجلد `images/` بالأسماء الموضحة في `images/README.txt`.
الموقع يدعم العرض حتى بدون الصور (يعرض أيقونات افتراضية).
